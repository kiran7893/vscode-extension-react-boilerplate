# Change Log

All notable changes to this extension will be documented in this file.

## [0.0.1] - 2024-10-29

### Added
- Initial release of VS Code React Extension Boilerplate
- TypeScript support for extension backend
- React 19 with modern hooks for webview UI
- Tailwind CSS integration with PostCSS processing
- esbuild for fast bundling (extension + webview)
- Production-ready build with minification
- CSS extraction and optimization
- Two-way messaging between extension and webview
- Watch mode for development
- VS Code theme-aware styling support

### Technical Details
- Extension bundle: ~2.5KB (minified)
- Webview bundle: ~192KB (minified, includes React + React-DOM)
- CSS bundle: ~8KB (minified, Tailwind utilities)
